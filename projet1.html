Projet : WM Collection

Description

WM Collection est un site e-commerce développé dans le cadre de ma formation. Il permet la vente de vêtements avec une interface client et une interface admin complète.

Fonctionnalités

  • ✅ Interface client avec panier, inscription et login
  • ✅ Interface admin pour gérer produits, commandes, utilisateurs
  • ✅ Intégration de base de données (MySQL)
  • ✅ Responsive Design avec CSS & Bootstrap

Technologies utilisées

HTML, CSS, JavaScript, PHP, MySQL
